### Install the libraries

```
$ sudo ./INSTALL
```

### Choose demo for MIPI camera or UVC

* MIPI: `detect_demo_mipi`
* UVC: `detect_demo_uvc`


E.g For UVC:

* Yoloface demo

```
$ ./detect_demo_uvc 0
```

* YoloV2 demo

```
$ ./detect_demo_uvc 1
```

* YoloV3 demo

```
$ ./detect_demo_uvc 2
```

### Uninstall the libraries

```
$ sudo ./UNINSTALL
```
